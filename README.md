# 🛒 FreshKart - AI Powered Grocery Delivery Platform

FreshKart is a full-stack grocery delivery application inspired by Blinkit and Instamart. It provides users with a modern shopping experience while offering administrators tools to manage products, inventory, and orders.

## 🚀 Features

### Customer

* User Registration & Login (JWT Authentication)
* Browse Products
* Product Detail Page
* Search Products
* Shopping Cart
* Wishlist
* Checkout
* Order History
* Profile Page
* Address Management

### Admin

* Dashboard
* Product Management
* Inventory Management
* Order Management
* Analytics
* Revenue Dashboard

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Swedhi/FreshKart.git
cd FreshKart
```

### Backend

```bash
cd backend
mvn spring-boot:run
```

Backend runs on:

```
http://localhost:8080
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```
